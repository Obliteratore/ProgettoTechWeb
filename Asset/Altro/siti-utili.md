# Siti utili all'accessibilità

## Siti raccomandati dalla prof

### **GRT Calculator**
Usato per otimizzare la reading experience degli utenti (Golden Ratio Typography).

https://grtcalculator.com/

### **Farfalla Project**
Usato per calcolare la comprensibilità di un testo, in questo caso è per l'indice Gulpease per un testo italiano.

https://farfalla-project.org/readability_static/

### **PerryMarshall**
Stessa cosa di Farfalla ma con testo in inglese.

https://www.perrymarshall.com/grade/

### **Estensione Chrome W3B1Y**
Estensione utilizzata per identificare eventuali problemi con l'accessibilità del sito.

### **Accesiblity Checking**
Sito che contiene tante informazioni sulla accessibilità, contiene una lista di font accessibili.

https://www.accessibilitychecking.com/blog/fonts-accessibility/

## Siti trovati online

### **axe DevTools**
Estensione per i browser che permette all'utente di fare test sull'accessibilità, include test automatici.

### **Accesiblity Insights**
Estensione per chrome ed edge per fare test sull'accessibilità (Microsoft)

https://accessibilityinsights.io/

### **Hue Sure**
Sito usato per controllare contrasti 

https://huesure.com

### **Colorsafe**
Trova pallette di colori accessibili.

http://colorsafe.co/

### **Tanaguru Contrast Finder**
"Find me the good constrasts, for web accessibility, between these two colors"

https://contrast-finder.tanaguru.com/
